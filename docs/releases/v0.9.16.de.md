Einem Speicher eines Freundes beizutreten erfordert jetzt keinen separaten Abstecher zu GitHub mehr, um dort etwas anzunehmen.

## Fehlerbehebungen

- 🤝 **Beitreten akzeptiert jetzt automatisch eine ausstehende Einladung** — eine GitHub-Mitarbeiter-Einladung muss explizit angenommen werden, bevor der Zugriff tatsächlich greift. "Mit einem Freund verbinden" prüfte bisher einfach den Zugriff und meldete "kein Zugriff — bitte deinen Freund, dich einzuladen", selbst direkt nachdem er das getan hatte, und ein Neustart der App half nicht, da es kein Caching-Problem war — GitHub selbst hatte den Zugriff einfach noch nicht gewährt. Jetzt wird nach einer ausstehenden Einladung zum Speicher des Hosts gesucht und diese automatisch angenommen, sodass der Beitritt einfach funktioniert, sobald der Host die Einladung gesendet hat.

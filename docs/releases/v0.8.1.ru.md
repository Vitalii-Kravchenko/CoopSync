Небольшое дополнение к автообновлениям из прошлого релиза — пара визуальных багов и новая настройка.

## Новое

- 🔔 **Переключатель "Автоматически проверять обновления"** в Настройках → Общие (включён по умолчанию). Выключи, чтобы пропустить тихую проверку в фоне вскоре после запуска — кнопка "Проверить обновления" в Настройках → О программе всё равно работает всегда.

## Исправления

- 🎯 Переключатели в Настройках (автозапуск, сворачивание в трей, предупреждение Steam Cloud, автопроверка обновлений) имели кружок немного не по центру, смещённый вниз и вбок — теперь точно по центру.
- 🔄 Кнопка "Проверить обновления" полностью исчезала, пока шла проверка — теперь остаётся на месте и просто блокируется до её завершения.

## What's new

- 🎮 **Subnautica 2 support** — save sync through the private GitHub repository, same as other games. Only the world's own save files are synced (no platform/account-specific files).
- ☁️ **Steam Cloud warning** — the app warns you if Steam Cloud might conflict with CoopSync's own sync for a game you're syncing.
- 🛡️ **Windows Smart App Control warning** — on Windows 11 this feature can block installing or running unsigned apps. There's now a warning in the README, on the installer's first screen, and in Settings → About, in all 10 interface languages.
- 🎨 Small UI fixes: sync notification icons are now our own (instead of emoji arrows), the app version shown in About is no longer hardcoded.

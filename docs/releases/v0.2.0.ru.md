## Что нового

- 🌍 **Мультиязычность — 10 языков интерфейса**: UA, EN, DE, FR, PL, RU, ES, PT-BR, TR, ZH-CN. Реактивный переключатель в Settings, локализованный трей (тултип + меню) без рестарта.
- 📦 **Мультиязычный NSIS-инсталлятор** — выбор языка при установке, выбранный язык сразу становится языком приложения после установки (даже при "установить для всех пользователей" с UAC-повышением).
- 🚀 **Автозапуск + трей по умолчанию** сразу после установки.
- 🎨 **Новый брендированный логотип/иконка** приложения (два пересекающихся круга — фирменная срезанная фаска).
- 🔧 Мелкие UI-фиксы: переключатели в Settings, размер карточек игр, иконки выбора роли host/join.

> Примечание: для этой исторической версии не сохранился корректный билд инсталлятора
> (тот, что собирался позже, уже содержал изменения из следующей версии) — только
> исходный код, без приложенного `.exe`.

Dołączenie do magazynu znajomego nie wymaga już osobnego wchodzenia na GitHub, żeby coś tam zaakceptować.

## Poprawki

- 🤝 **Dołączanie teraz samo akceptuje oczekujące zaproszenie** — zaproszenie do współpracy na GitHubie trzeba jawnie zaakceptować, zanim dostęp faktycznie zadziała. „Połącz się ze znajomym" wcześniej po prostu sprawdzało dostęp i pisało „brak dostępu — poproś znajomego, żeby cię zaprosił", nawet zaraz po tym, jak faktycznie to zrobił, a ponowne uruchomienie aplikacji nie pomagało, bo to nie był problem z pamięcią podręczną — sam GitHub jeszcze nie przyznał dostępu. Teraz aplikacja sprawdza, czy jest oczekujące zaproszenie do magazynu hosta, i akceptuje je automatycznie, więc dołączenie po prostu działa, gdy tylko host wyśle zaproszenie.

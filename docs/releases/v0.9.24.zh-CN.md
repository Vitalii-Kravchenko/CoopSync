日常使用自定义游戏时发现的七个修复。

## 修复

- 🌐 **安装程序现在会使用你的 Windows 语言** — 安装前的 Smart App Control 提示虽然已翻译成全部 10 种语言，但之前总是显示英文。现在会自动匹配你的 Windows 显示语言，如果你的语言不在这十种之内，则回退到英文。
- 🎯 **自定义游戏选择的 .exe 现在真正会保持选中** — 重新打开详情页时完全不显示已选中的可执行文件，尽管它其实已正确保存。这只是一个显示上的问题，让人误以为选择丢失了。
- 🏷️ **"手动添加"现在显示在游戏名称旁边**，而不是下方单独一行。
- 🖼️ **文件选择对话框（封面、头像）现在会跟随你选择的语言** — 之前无论设置里选了什么，始终显示为乌克兰语。
- 🙂 **你自己的头像现在能在完全重装后保留下来** — 之前卸载再重装后，头像会在各处消失（无论是你自己还是好友那边看到的），尽管它其实一直安全地存在 GitHub 上。现在会自动恢复。
- 🗂️ **添加自定义游戏时就可以直接排除文件** — 不用再先添加游戏，之后再到详情里找这个选项。
- ☁️ **封面未能同步到 GitHub 时现在会提示你** — 添加自定义游戏或更改其封面时，可能会悄悄同步失败而队友完全不知情。现在会显示警告并带有"重试"按钮，而不是默默消失。

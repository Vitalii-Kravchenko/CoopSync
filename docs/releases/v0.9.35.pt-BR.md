Uma única correção para uma entrada duplicada de inicialização automática que algumas instalações acumularam com o tempo.

## Correções

- 🚀 **O CoopSync não inicia mais duas vezes com o Windows.** Uma versão mais antiga (antes de o app ter uma identidade explícita para o Windows) registrava sua entrada de inicialização automática com um nome padrão gerado automaticamente. Uma versão posterior passou a gerenciar a inicialização automática usando a identidade real do app, mas nunca removeu aquela entrada antiga — então as duas disparavam a cada login, iniciando um segundo processo redundante (fechado imediatamente, mas ainda visível como uma entrada extra na lista de apps de inicialização do Windows e no Gerenciador de Tarefas). Esta atualização remove a entrada obsoleta automaticamente na primeira execução.

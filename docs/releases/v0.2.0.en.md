## What's new

- 🌍 **Multi-language UI — 10 languages**: UA, EN, DE, FR, PL, RU, ES, PT-BR, TR, ZH-CN. Reactive switcher in Settings, localized tray (tooltip + menu) with no restart needed.
- 📦 **Multi-language NSIS installer** — pick a language during install, it becomes the app's language right after installation (even with "install for all users" + UAC elevation).
- 🚀 **Auto-start + tray enabled by default** right after install.
- 🎨 **New branded app logo/icon** (two overlapping circles — signature angled corner cut).
- 🔧 Small UI fixes: Settings toggles, game card size, host/join role icons.

> Note: no accurate installer build is preserved for this historical version (the one built
> later already contained changes from the next version) — source code only, no attached `.exe`.

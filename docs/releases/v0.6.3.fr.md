- 👁️ Amélioration du contraste de ce texte pour qu'il reste lisible sur n'importe quelle jaquette de jeu, quelle que soit la luminosité de l'illustration.
- 🐛 Correction de l'absence totale de l'affichage "dernière synchro" / "taille des sauvegardes" pour les jeux installés listés dans "Tous les jeux pris en charge" — il s'affiche maintenant là aussi, comme dans la section "Jeux installés" ci-dessus.

Un parche centrado en el manejo de errores — sin funciones nuevas, solo mensajes más claros cuando algo falla, y algunos puntos que antes se quedaban atascados o mostraban un estado engañoso ya no lo hacen.

## Correcciones

- 🌐 **Mensajes claros de "sin conexión a internet" para acciones de GitHub** (inicio de sesión, creación del almacenamiento, invitar a un amigo) — algunos de ellos antes mostraban un error técnico en bruto en su lugar.
- ⏳ La aplicación ya no se queda atascada para siempre en "Cargando…" si una comprobación de red falla justo al iniciar — ahora muestra un error claro con un botón "Inténtalo de nuevo".
- 🔑 Un fallo de red temporal al comprobar la sesión ya no te cierra la sesión — ahora solo lo hace una sesión realmente caducada.
- 📦 Ajustes, Amigos y la pantalla de configuración inicial ya no afirman falsamente que el almacenamiento no está configurado cuando simplemente falló una comprobación de red — ahora muestran el error real con opción de reintentar.
- 🎮 La lista de juegos ya no se queda atascada para siempre en "Buscando juegos…" si algo falla al cargarla.
- 🔄 La sincronización automática ya no deja de funcionar en silencio cuando no puede comprobar qué juegos están abiertos — ahora verás un mensaje en lugar de que se apague sin más.
- 📡 El inicio de sesión ahora sobrevive a un corte de red breve en lugar de interrumpir todo el proceso.
- ⏱️ Nuevo mensaje cuando se alcanza temporalmente el límite de solicitudes de la API de GitHub, indicando cuándo volver a intentarlo.

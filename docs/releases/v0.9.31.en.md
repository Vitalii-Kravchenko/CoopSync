Found it, with live proof this time: whoever originally added a custom game could bring it right back after anyone else deleted it.

## Fixes

- 🗑️ **Deleting a custom game now actually sticks, no matter who added it originally.** Caught red-handed: the exact same game got pushed back within seconds of being deleted, twice in a row. Turns out any custom game can be removed by either person, but the person who originally added it had no way to tell "someone else just deleted this" apart from "my own upload of it never went through" — so their app kept guessing the second one and undoing the deletion. It now remembers whether a game was ever actually confirmed as shared before, and only treats a missing one as a genuine deletion to respect once it has been.

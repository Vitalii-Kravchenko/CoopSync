Um patch focado em tratamento de erros — sem novidades, apenas mensagens mais claras quando algo dá errado, e alguns pontos que antes travavam ou mostravam um status enganoso agora não fazem mais isso.

## Correções

- 🌐 **Mensagens claras de "sem conexão com a internet" para ações do GitHub** (login, criação do armazenamento, convite de amigo) — algumas delas antes mostravam um erro técnico bruto em vez disso.
- ⏳ O aplicativo não fica mais travado para sempre em "Carregando…" se uma verificação de rede falhar bem na inicialização — agora mostra um erro claro com um botão "Tentar novamente".
- 🔑 Uma falha temporária de rede ao verificar a sessão não te desconecta mais — só uma sessão realmente expirada faz isso agora.
- 📦 Configurações, Amigos e a tela de configuração inicial não afirmam mais falsamente que o armazenamento não foi criado quando uma verificação de rede simplesmente falhou — agora mostram o erro real com opção de tentar novamente.
- 🎮 A lista de jogos não fica mais travada para sempre em "Procurando jogos…" se algo der errado ao carregar.
- 🔄 A sincronização automática não para mais de funcionar silenciosamente quando não consegue verificar quais jogos estão em execução — agora você verá uma mensagem em vez dela simplesmente sumir.
- 📡 O login agora sobrevive a uma breve falha de rede em vez de interromper todo o processo de entrada.
- ⏱️ Nova mensagem quando o limite de requisições da API do GitHub é atingido temporariamente, avisando quando tentar novamente.

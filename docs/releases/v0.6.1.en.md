A reliability-focused patch — no new features, just fixes for sync safety, a startup crash, and a few UI rough edges.

## Fixes

- 🖥️ **The app failed to open at all on some GPUs** (older AMD cards especially) — hardware acceleration is now disabled unconditionally, so rendering always falls back to software.
- 💾 **Deleted or missing local saves are now restored automatically** when you launch a game — CoopSync fills in whatever's missing from the cloud without touching files you already have, instead of silently doing nothing.
- 🛡️ **Old backups no longer overwrite newer cloud saves** — if your local save hasn't changed since the last sync but looks different from the cloud (e.g. you restored an old backup), CoopSync now recognizes it as stale, pulls the cloud version automatically, and skips auto-uploading it over your friend's progress.
- 🔢 Fixed a bug where uploading created a brand-new save version even when nothing had actually changed.
- 🔁 Version numbers now roll over naturally (v1.999 → v2.000) instead of climbing forever.
- 🎨 Fixed the sync-status badge clipping its own background when the text wrapped to two lines, and centered its status dot.
- 🔘 The "change role" link on the onboarding screen is now a proper small button instead of a bare underlined text link.
- 🖼️ Game poster art now refreshes once per app launch instead of staying cached forever, so updated Steam artwork shows up sooner.

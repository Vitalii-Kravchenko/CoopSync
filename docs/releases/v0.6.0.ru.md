# CoopSync v0.6.0

[English](v0.6.0.en.md) · [Français](v0.6.0.fr.md) · [Deutsch](v0.6.0.de.md) · [Polski](v0.6.0.pl.md) · [Português (Brasil)](v0.6.0.pt-BR.md) · **Русский** · [简体中文](v0.6.0.zh-CN.md) · [Español](v0.6.0.es.md) · [Türkçe](v0.6.0.tr.md) · [Українська](v0.6.0.uk.md)

## Новое

- 🎮 **Переработан ховер карточки игры** — наведение на установленную игру теперь показывает крупные кнопки «Выгрузить»/«Загрузить» с подписями, а также время последней синхронизации и размер сейвов, вместо двух маленьких icon-only кнопок.
- ✨ Унифицированы тайминги hover/transition-анимаций по всему приложению — карточки, кнопки, переключатели и боковая панель теперь двигаются с одинаковой скоростью, а не слегка по-разному.

## Исправления

- 🌍 **Все сообщения об ошибках теперь полностью локализованы** — ошибки GitHub, синхронизации и хранилища раньше всегда показывались на украинском независимо от выбранного языка. Теперь каждое сообщение переведено на все 10 поддерживаемых языков.
- 🔇 Исправлено несколько мест, где неудачное действие не показывало вообще ничего — приглашение друга, загрузка истории синхронизации и проверка статуса синхронизации могли упасть без единого отклика. Теперь они показывают понятную ошибку (а для статуса синхронизации — ещё и кнопку «Попробовать снова»).
- 🔁 Приложение теперь само восстанавливается, если локальная папка синхронизации устарела (например, после удаления и пересоздания общего репозитория), вместо того чтобы вечно падать с технической ошибкой git.
- 🐛 Исправлена путаница: приглашение несуществующего юзера GitHub показывало «хранилище не найдено» вместо «пользователь не найден».
- ➕ В Настройках теперь появляется кнопка «Создать хранилище», когда общее хранилище не настроено (например, сразу после удаления) — раньше там была только надпись «хранилище не настроено» без какого-либо способа это исправить.

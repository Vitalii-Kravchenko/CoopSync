## Novedades

- 🌍 **Interfaz multilingüe — 10 idiomas**: UA, EN, DE, FR, PL, RU, ES, PT-BR, TR, ZH-CN. Selector reactivo en Ajustes, bandeja del sistema localizada (información + menú) sin reiniciar.
- 📦 **Instalador NSIS multilingüe** — elige el idioma durante la instalación, que se convierte de inmediato en el idioma de la aplicación tras instalar (incluso con "instalar para todos los usuarios" y elevación UAC).
- 🚀 **Inicio automático + bandeja activados de forma predeterminada** justo después de instalar.
- 🎨 **Nuevo logotipo/icono de marca** de la aplicación (dos círculos superpuestos — el característico corte angular).
- 🔧 Pequeñas correcciones de interfaz: interruptores en Ajustes, tamaño de las tarjetas de juegos, iconos de selección de rol anfitrión/invitado.

> Nota: no se conserva una compilación fiel del instalador para esta versión histórica
> (la construida después ya contenía cambios de la siguiente versión) — solo código
> fuente, sin `.exe` adjunto.

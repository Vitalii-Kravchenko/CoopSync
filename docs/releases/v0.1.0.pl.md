Pierwsze działające MVP CoopSync — od zera do w pełni funkcjonalnej aplikacji w kilka sesji (29.06–01.07.2026).

## Co zawiera

- 🔑 Logowanie przez GitHub (device flow) — bez hasła, bez przekierowania OAuth
- 🔒 Prywatne wspólne repozytorium zapisów (`coopsync-saves`) — zaproszenie znajomego jako współpracownika, lista uczestników z automatycznym sprawdzaniem statusu akceptacji
- 🎮 Wykrywanie zainstalowanych gier Steam (rejestr + `libraryfolders.vdf`)
- 🖥️ Pełny interfejs: onboarding, ekran główny z kartami gier, ustawienia, własny pasek tytułu i panel boczny
- 🔄 Rdzeń synchronizacji — ręczny upload/download, wersje zapisów, status synchronizacji (synced/local-newer/remote-newer itd.)
- ⚙️ Automatyczna synchronizacja — śledzi uruchomienie/zamknięcie gry w tle, automatyczny pull/push
- 🗂️ Działanie w tle: zasobnik systemowy, autostart z Windows
- 👑 Host/join — wybór roli (jestem hostem / łączę się ze znajomym), synchronizacja ze wspólnym repozytorium
- 🎨 System projektowy **RIFT//SYNC (Kinetic Edge)** zastosowany w całym interfejsie

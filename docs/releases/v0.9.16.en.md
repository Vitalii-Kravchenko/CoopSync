Joining a friend's storage no longer needs a separate trip to GitHub to accept anything.

## Fixes

- 🤝 **Joining now auto-accepts a pending invite** — a GitHub collaborator invite has to be explicitly accepted before access actually kicks in. "Connect to a friend" used to just check for access and say "no access — ask your friend to invite you" even right after they had, and restarting the app didn't help since it wasn't a caching issue — GitHub itself hadn't granted access yet. Now it checks for a pending invite to the host's storage and accepts it automatically, so joining just works as soon as the host has sent the invite.

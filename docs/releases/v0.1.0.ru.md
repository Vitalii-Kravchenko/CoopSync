# CoopSync v0.1.0

[English](v0.1.0.en.md) · [Français](v0.1.0.fr.md) · [Deutsch](v0.1.0.de.md) · [Polski](v0.1.0.pl.md) · [Português (Brasil)](v0.1.0.pt-BR.md) · **Русский** · [简体中文](v0.1.0.zh-CN.md) · [Español](v0.1.0.es.md) · [Türkçe](v0.1.0.tr.md) · [Українська](v0.1.0.uk.md)

Первый рабочий MVP CoopSync — с нуля до полностью функционального приложения за несколько занятий (29.06–01.07.2026).

## Что есть

- 🔑 Вход через GitHub (device flow) — без пароля, без OAuth-редиректа
- 🔒 Приватный общий репозиторий сейвов (`coopsync-saves`) — приглашение друга в соавторы, список участников с автоопросом статуса принятия
- 🎮 Детект установленных Steam-игр (реестр + `libraryfolders.vdf`)
- 🖥️ Полный UI: онбординг, главный экран с карточками игр, настройки, собственный titlebar и боковая панель
- 🔄 Ядро синхронизации — ручной upload/download, версии сейвов, статус синка (synced/local-newer/remote-newer и т.д.)
- ⚙️ Автосинхронизация — слежение за запуском/выходом игры в фоне, авто-pull/push
- 🗂️ Работа в фоне: трей, автозапуск вместе с Windows
- 👑 Host/join — выбор роли (я главный / подключаюсь к другу), синхронизация с общим репозиторием
- 🎨 Дизайн-система **RIFT//SYNC (Kinetic Edge)** применена ко всему интерфейсу

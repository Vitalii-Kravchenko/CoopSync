# CoopSync v0.1.0

[English](v0.1.0.en.md) · [Français](v0.1.0.fr.md) · **Deutsch** · [Polski](v0.1.0.pl.md) · [Português (Brasil)](v0.1.0.pt-BR.md) · [Русский](v0.1.0.ru.md) · [简体中文](v0.1.0.zh-CN.md) · [Español](v0.1.0.es.md) · [Türkçe](v0.1.0.tr.md) · [Українська](v0.1.0.uk.md)

Das erste funktionierende CoopSync-MVP — von Grund auf zu einer voll funktionsfähigen App in wenigen Sitzungen (29.06.–01.07.2026).

## Was enthalten ist

- 🔑 GitHub-Login (Device Flow) — kein Passwort, keine OAuth-Weiterleitung
- 🔒 Privates gemeinsames Speicher-Repository (`coopsync-saves`) — Freund als Mitwirkenden einladen, Mitgliederliste mit automatischer Statusabfrage
- 🎮 Erkennung installierter Steam-Spiele (Registry + `libraryfolders.vdf`)
- 🖥️ Vollständige UI: Onboarding, Hauptbildschirm mit Spielkarten, Einstellungen, eigene Titelleiste und Seitenleiste
- 🔄 Sync-Kern — manuelles Hoch-/Herunterladen, Speicherversionen, Sync-Status (synced/local-newer/remote-newer usw.)
- ⚙️ Auto-Sync — überwacht im Hintergrund Spielstart/-ende, automatisches Pull/Push
- 🗂️ Hintergrundbetrieb: System-Tray, Autostart mit Windows
- 👑 Host/Join — Rolle wählen (Ich bin der Host / Verbindung zu einem Freund), Synchronisierung mit dem gemeinsamen Repository
- 🎨 Designsystem **RIFT//SYNC (Kinetic Edge)** auf die gesamte UI angewendet

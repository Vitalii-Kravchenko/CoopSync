## What's new

- 💬 **New "Support" button in the top bar** — send feedback, bug reports, or feature requests straight from the app, no need to email me directly.
- 🎮 **"Want a game added" now searches Steam live** — pick up to 3 games by name (with cover art) instead of typing free text, so requests are clear and easy to act on.
- 💡 **New "Have an idea" category** — for anything beyond bugs and game requests.
- ✉️ Feedback emails now include your GitHub name, clickable Steam links for requested games, and read a lot nicer overall.

## Fixes

- ⌨️ **Accessibility fixes** — dialogs (Support, delete confirmation) now properly trap keyboard focus instead of leaking into the app behind them, and focus indicators work correctly everywhere, including the app's signature angle-cut buttons.

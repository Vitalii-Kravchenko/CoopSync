Первый шаг к истории отката версий для каждой игры отдельно.

## Новое

- 🕓 На карточке каждой игры теперь есть кнопка **Подробнее**, которая открывает отдельный экран с историей синхронизаций именно этой игры — отдельно от общей истории на вкладке "История".
- 🧭 Навигация-хлебные крошки (`Игры / Название игры → История синхронизаций`) для возврата в библиотеку.

Откат на более старую версию сейва с этого экрана появится в будущем обновлении — пока это только просмотр истории.

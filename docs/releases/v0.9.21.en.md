Any game you own can sync now — not just the ones in the built-in catalog, and not just at the guessed default path.

## New

- 🗂️ **Edit any game's save folder from its detail screen** — if CoopSync guessed the wrong path (a non-Steam install, a moved user folder, etc.) you can now correct it yourself instead of being stuck. Open a game's detail screen → "Save location" → Edit, browse to the real folder, save. "Reset to default" brings back the catalog's guess.
- ➕ **Add a game outside the built-in catalog** — a "+ Add a game" card now sits at the top of Installed games. Point it at the save folder and, optionally, the game's install folder — CoopSync scans that for the .exe automatically and enables the same launch/exit auto-sync a catalog game gets (or pick the .exe by hand if the scan misses it). Skip the install folder and it still works, just manual Upload/Download. Since we don't know a custom game's save structure, the whole folder syncs as-is, unlike the filtering catalog games get.
- 🤝 **A custom game you add shows up for your co-op partner too** — pushed to a small shared registry, so their app picks it up automatically and asks them to point it at their own save folder before it starts syncing on their end.

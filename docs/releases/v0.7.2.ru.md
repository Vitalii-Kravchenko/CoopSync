Патч на надёжность обработки ошибок — без новых фич, только более понятные сообщения, когда что-то идёт не так, и несколько мест, которые раньше зависали или показывали обманчивый статус, теперь этого не делают.

## Исправления

- 🌐 **Чёткие сообщения "нет интернета" для действий с GitHub** (вход, создание хранилища, приглашение друга) — раньше некоторые из них показывали сырую техническую ошибку вместо этого.
- ⏳ Приложение больше не зависает навечно на "Загрузка…", если проверка сети падает прямо на старте — теперь показывает чёткую ошибку с кнопкой "Попробовать снова".
- 🔑 Временный сбой сети при проверке сессии больше не разлогинивает тебя — только по-настоящему истёкшая сессия делает это теперь.
- 📦 Настройки, Друзья и экран начальной настройки больше не врут, что хранилище не создано, когда просто упала проверка сети — теперь показывают реальную ошибку с возможностью попробовать снова.
- 🎮 Список игр больше не зависает навечно на "Ищу игры…", если что-то пошло не так при загрузке.
- 🔄 Автосинхронизация больше не перестаёт молча работать, если не удаётся проверить, какие игры запущены — теперь увидишь сообщение вместо тихого исчезновения.
- 📡 Вход теперь переживает кратковременный сбой сети вместо обрыва всего процесса входа.
- ⏱️ Новое сообщение, когда временно исчерпан лимит запросов к GitHub API, с подсказкой, когда попробовать снова.

First step toward per-game rollback history.

## New

- 🕓 Each game's card now has a **Details** button that opens a dedicated screen with that game's own sync history — separate from the shared history on the History tab.
- 🧭 Breadcrumb navigation (`Games / Game name → Sync history`) to get back to your library.

Rolling back to an older save version from this screen is coming in a future update — for now it's a read-only history view.

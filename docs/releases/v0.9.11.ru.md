Шесть исправлений из беклога — трей-уведомления, обратная связь при восстановлении и гонка с фоновым автопушем.

## Исправления

- 🔔 **Тост об обновлении теперь действительно открывает приложение** — клик по уведомлению Windows «доступно обновление» ничего не делал, пока CoopSync сидел скрытым в трее. Теперь это обрабатывает само приложение напрямую, а не окно, которое могло ещё не «слушать».
- 🔕 **Больше никаких дублей уведомлений об обновлении** — выход и повторный запуск через трей, пока та же версия оставалась самой новой, каждый раз добавляли новую запись в колокольчик. Теперь это запоминается как положено.
- ⏳ **Восстановление показывает, что действительно работает** — восстановление старой версии сейва раньше просто делало кнопки серыми, без других признаков процесса. Теперь там крутилка.
- ⏳ **Кадрирование аватарки ждёт настоящей загрузки** — нажатие «Применить» после кадрирования закрывало окно сразу, ещё до того, как загрузка реально завершилась. Теперь окно остаётся на месте с крутилкой и подтверждает баннером, когда всё действительно готово.
- ✖️ **Более чистые быстрые уведомления** — убрал бесполезную кнопку закрытия с маленьких уведомлений о синхронизации; они и так исчезают сами через 5 секунд.
- 🔒 **Восстановление больше не конфликтует с фоновым автопушем** — сразу после выхода из игры CoopSync тихо пушит сейв в фоне. Кнопка «Восстановить» для этой игры теперь заблокирована, пока этот пуш реально не завершится, чтобы они не боролись за одни и те же данные.

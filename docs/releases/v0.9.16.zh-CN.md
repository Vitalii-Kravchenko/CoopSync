加入好友的存储不再需要单独跑一趟 GitHub 去接受什么邀请了。

## 修复

- 🤝 **加入时现在会自动接受待处理的邀请** — GitHub 的协作者邀请必须先被明确接受，访问权限才会真正生效。以前"连接到好友"只是简单检查一下访问权限，然后提示"没有访问权限——请让好友邀请你"，哪怕好友其实刚刚邀请过；重启应用也没用，因为这根本不是缓存问题——GitHub 那边压根还没授予访问权限。现在它会检查主机存储是否有待处理的邀请，并自动接受，这样只要主机发送了邀请，加入操作就能直接生效。

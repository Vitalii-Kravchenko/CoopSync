Ein Zuverlässigkeits-Patch — keine neuen Funktionen, nur Fixes für Sync-Sicherheit, einen Absturz beim Start und ein paar UI-Kanten.

## Fehlerbehebungen

- 🖥️ **Die App ließ sich auf manchen GPUs überhaupt nicht öffnen** (besonders ältere AMD-Karten) — Hardwarebeschleunigung ist jetzt bedingungslos deaktiviert, das Rendering läuft immer per Software.
- 💾 **Gelöschte oder fehlende lokale Spielstände werden jetzt automatisch wiederhergestellt**, wenn du ein Spiel startest — CoopSync ergänzt, was aus der Cloud fehlt, ohne bereits vorhandene Dateien anzurühren, statt einfach stillschweigend nichts zu tun.
- 🛡️ **Alte Backups überschreiben keine neueren Cloud-Spielstände mehr** — wenn sich dein lokaler Spielstand seit der letzten Synchronisierung nicht geändert hat, aber anders aussieht als der Cloud-Stand (z. B. weil ein altes Backup eingespielt wurde), erkennt CoopSync das jetzt als veraltet, lädt automatisch die Cloud-Version herunter und überspringt den automatischen Upload, statt den Fortschritt deines Freundes zu überschreiben.
- 🔢 Ein Fehler behoben, bei dem ein Upload eine völlig neue Speicherstand-Version erstellte, obwohl sich eigentlich nichts geändert hatte.
- 🔁 Versionsnummern laufen jetzt natürlich über (v1.999 → v2.000), statt endlos weiterzuzählen.
- 🎨 Ein Fehler behoben, bei dem das Sync-Status-Badge seinen eigenen Hintergrund abschnitt, wenn der Text auf zwei Zeilen umbrach, und der Status-Punkt vertikal zentriert.
- 🔘 Der Link „Rolle ändern" im Onboarding ist jetzt ein richtiger kleiner Button statt eines nackten, unterstrichenen Textlinks.
- 🖼️ Spiele-Cover werden jetzt einmal pro App-Start neu geladen, statt für immer zwischengespeichert zu bleiben — aktualisierte Steam-Artworks erscheinen dadurch schneller.

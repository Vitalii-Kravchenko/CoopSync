Poprawka skupiona na obsłudze błędów — brak nowych funkcji, tylko jaśniejsze komunikaty, gdy coś pójdzie nie tak, i kilka miejsc, które wcześniej zawieszały się lub pokazywały mylący status, już tego nie robi.

## Poprawki

- 🌐 **Jasne komunikaty "brak połączenia z internetem" dla działań związanych z GitHub** (logowanie, tworzenie magazynu, zapraszanie znajomego) — niektóre z nich wcześniej pokazywały surowy techniczny błąd zamiast tego.
- ⏳ Aplikacja już nie wisi wiecznie na "Ładowanie…", gdy sprawdzenie sieci zawiedzie tuż przy starcie — teraz pokazuje wyraźny błąd z przyciskiem "Spróbuj ponownie".
- 🔑 Tymczasowy problem z siecią podczas sprawdzania sesji już cię nie wylogowuje — robi to teraz tylko naprawdę wygasła sesja.
- 📦 Ustawienia, Znajomi i ekran konfiguracji już nie kłamią, że magazyn nie jest skonfigurowany, gdy po prostu zawiodło sprawdzenie sieci — teraz pokazują prawdziwy błąd z możliwością ponownej próby.
- 🎮 Lista gier już nie wisi wiecznie na "Szukam gier…", jeśli coś pójdzie nie tak przy ładowaniu.
- 🔄 Automatyczna synchronizacja już po cichu nie przestaje działać, gdy nie da się sprawdzić, które gry są uruchomione — teraz zobaczysz komunikat zamiast cichego zaniku.
- 📡 Logowanie przetrwa teraz krótką przerwę w sieci zamiast przerywać cały proces logowania.
- ⏱️ Nowy komunikat, gdy tymczasowo wyczerpany zostanie limit żądań GitHub API, z informacją, kiedy spróbować ponownie.

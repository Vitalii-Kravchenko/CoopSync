## Що нового

- 🌍 **Мультимовність — 10 мов інтерфейсу**: UA, EN, DE, FR, PL, RU, ES, PT-BR, TR, ZH-CN. Реактивний перемикач у Settings, локалізований трей (тултип + меню) без рестарту.
- 📦 **Мультимовний NSIS-інсталятор** — вибір мови на встановленні, обрана мова одразу стає мовою застосунку після інсталяції (навіть при "встановити для всіх користувачів" з UAC-елевацією).
- 🚀 **Автозапуск + трей за замовчуванням** одразу після встановлення.
- 🎨 **Новий брендований логотип/іконка** застосунку (два кола, що перетинаються, — фірмова зрізана фаска).
- 🔧 Дрібні UI-фікси: тумблери в Settings, розмір карток ігор, іконки вибору ролі host/join.

> Примітка: для цієї історичної версії немає збереженого коректного білда інсталятора
> (той, що збирався пізніше, вже містив зміни з наступної версії) — тому без вкладеного
> `.exe`, тільки вихідний код тегу.

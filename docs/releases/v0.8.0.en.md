The app can now update itself — this is the last release you'll ever need to download by hand.

## What's new

- 🔔 **In-app updates.** CoopSync checks for a new release in the background shortly after launch, and there's also a "Check for updates" button in Settings → About.
- ⬇️ Downloading and installing are separate steps you control — nothing happens automatically. When an update is found you choose to download it, then choose to restart and install it.
- 📉 Updates only fetch the parts of the installer that actually changed, not the whole ~100 MB file every time.
- 🔵 A small dot appears on the Settings icon in the sidebar when an update is ready, so it's easy to notice even if you never open Settings.

**One-time note:** since this is the first version with the updater built in, upgrading from v0.7.4 or earlier still needs a manual download — grab the installer below. Every release after this one will offer itself automatically.

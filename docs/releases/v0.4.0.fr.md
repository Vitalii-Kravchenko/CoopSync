## Nouveautés

- 👥 **Onglet Amis** — la gestion des amis a quitté les Paramètres pour son propre écran dans la barre latérale. Invite par nom d'utilisateur GitHub et vois qui a accepté, qui est encore en attente.
- 🗑️ **Suppression du dépôt** — une nouvelle action sensible dans les Paramètres pour supprimer définitivement le stockage partagé : boîte de confirmation avec un compte à rebours de 10 secondes avant que le bouton de suppression ne se débloque.

## Corrections

- 🐛 Correction d'un bug de synchronisation : quitter un jeu poussait toujours ta sauvegarde locale sur GitHub, même si un ami avait déjà poussé une version plus récente pendant que vous jouiez ensemble — écrasant silencieusement sa progression. CoopSync vérifie désormais d'abord la version dans le cloud et saute le push (avec un avertissement) si elle est déjà plus récente.
- 🐛 Les échecs de synchronisation en arrière-plan passaient auparavant inaperçus, sans aucun retour — ils s'affichent désormais aussi dans une bannière.

## Autres changements

- 📘 README : remplacement des badges stylisés à la main par de vrais badges shields.io (GitHub supprime les attributs `style` en ligne du HTML des README, donc les anciens s'affichaient en texte brut), un tableau des prérequis plus clair, et une note explicite indiquant que Git pour Windows est la seule chose à installer soi-même.

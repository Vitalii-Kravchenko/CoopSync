Agora o aplicativo pode se atualizar sozinho — essa é a última vez que você vai precisar baixar um instalador manualmente.

## Novidades

- 🔔 **Atualizações dentro do aplicativo.** O CoopSync verifica em segundo plano se há uma nova versão pouco depois de abrir, e também há um botão "Verificar atualizações" em Configurações → Sobre.
- ⬇️ Baixar e instalar são etapas separadas que você controla — nada acontece automaticamente. Quando uma atualização é encontrada, você escolhe baixá-la, e depois escolhe reiniciar e instalar.
- 📉 As atualizações baixam apenas as partes do instalador que realmente mudaram, não os ~100 MB inteiros toda vez.
- 🔵 Um pequeno ponto aparece no ícone de Configurações na barra lateral quando uma atualização está pronta — fácil de notar mesmo que você nunca abra Configurações.

**Aviso único:** como esta é a primeira versão com o atualizador embutido, atualizar a partir da v0.7.4 ou anterior ainda exige download manual — pegue o instalador abaixo. Todas as versões seguintes vão se oferecer sozinhas.

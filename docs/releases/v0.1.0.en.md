The first working CoopSync MVP — built from scratch to a fully functional app over a few sessions (Jun 29 – Jul 1, 2026).

## What's included

- 🔑 GitHub login (device flow) — no password, no OAuth redirect
- 🔒 Private shared saves repository (`coopsync-saves`) — invite a friend as a collaborator, member list with auto-polling for acceptance status
- 🎮 Detection of installed Steam games (registry + `libraryfolders.vdf`)
- 🖥️ Full UI: onboarding, main screen with game cards, settings, custom titlebar and sidebar
- 🔄 Sync core — manual upload/download, save versions, sync status (synced/local-newer/remote-newer, etc.)
- ⚙️ Auto-sync — watches for the game starting/quitting in the background, auto pull/push
- 🗂️ Background operation: system tray, auto-start with Windows
- 👑 Host/join — choose your role (I'm the host / connecting to a friend), sync with the shared repository
- 🎨 **RIFT//SYNC (Kinetic Edge)** design system applied across the whole UI

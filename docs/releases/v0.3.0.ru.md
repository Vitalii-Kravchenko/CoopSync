## Что нового

- 🎮 **Поддержка Subnautica 2** — синхронизация сейвов через приватный GitHub-репозиторий, как и для других игр. Синхронизируются только файлы самого мира (без аккаунт-специфичных файлов платформы).
- ☁️ **Предупреждение о Steam Cloud** — приложение предупреждает, если Steam Cloud может конфликтовать с собственной синхронизацией CoopSync для синхронизируемой игры.
- 🛡️ **Предупреждение о Windows Smart App Control** — на Windows 11 эта функция может заблокировать установку или запуск неподписанных программ. Теперь предупреждение есть в README, на первом экране инсталлятора и в Настройках → О программе, на всех 10 языках интерфейса.
- 🎨 Мелкие UI-фиксы: иконки синка в уведомлениях теперь свои (вместо эмодзи-стрелок), версия приложения в "О программе" больше не захардкожена.

Updates now happen quietly in the background — plus custom games got a lot more capable this round.

## New

- 🔕 **Updates install silently now** — no more installer window popping up when you restart to apply one; it just happens in the background, then restarts. Fresh installs are simpler too — just run the installer, no wizard.
- ⚠️ **A heads-up before install if Smart App Control would block it** — replacing the old installer warning page, setup now checks first and tells you how to turn it off instead of failing silently.
- 🎯 **Set up a custom game's install folder from its detail screen, not just when adding it** — a co-op partner completing setup for a game they didn't add themselves could only fix the save path before; the same install-folder/.exe picker from "Add a game" is right there now too.
- 🤝 **A custom game's cover art now syncs to your co-op partner** — shows up on their side automatically, without ever overwriting a cover they already picked themselves.
- 🗂️ **Exclude specific files from a custom game's sync** — if its save folder also has local settings or account files mixed in with the actual save data, pick which files to leave out, right from its detail screen.

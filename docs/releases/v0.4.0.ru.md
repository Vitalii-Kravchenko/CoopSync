## Новое

- 👥 **Вкладка «Друзья»** — управление друзьями переехало из Настроек в отдельный экран в боковой панели. Приглашай по нику GitHub, смотри кто принял приглашение, а кто ещё ожидает.
- 🗑️ **Удаление репозитория** — новое опасное действие в Настройках: навсегда удалить общее хранилище сейвов, с подтверждением и 10-секундным отсчётом, прежде чем кнопка удаления разблокируется.

## Исправления

- 🐛 Исправлен баг синхронизации: выход из игры всегда пушил локальный сейв на GitHub, даже если друг уже запушил более новую версию, пока вы играли вместе — это молча затирало его прогресс. Теперь CoopSync сначала проверяет версию в облаке и пропускает push (с предупреждением), если она уже новее.
- 🐛 Неудачные фоновые синхронизации раньше молча пропадали без единого сообщения — теперь тоже показываются баннером.

## Другие изменения

- 📘 README: самописные бейджи заменены на настоящие shields.io (GitHub вырезает инлайн `style` из HTML в README, поэтому старые отображались голым текстом), более понятная таблица требований и явное примечание, что Git для Windows — единственное, что нужно ставить самому.

# CoopSync v0.4.0

## 新功能

- 👥 **好友标签页** — 好友管理从设置中移出，成为侧边栏中独立的界面。通过 GitHub 用户名邀请好友，查看谁已接受、谁还在等待。
- 🗑️ **删除仓库** — 设置中新增的危险操作，用于永久删除共享存档存储：确认对话框带有 10 秒倒计时，之后删除按钮才会解锁。

## 修复

- 🐛 修复了一个同步错误：退出游戏时总会把本地存档推送到 GitHub，即使你们一起玩的时候朋友已经推送了更新的版本——这会悄悄覆盖他的进度。现在 CoopSync 会先检查云端版本，如果云端已经更新，就跳过推送（并显示警告）。
- 🐛 之前后台同步失败时会悄无声息、没有任何提示——现在也会以横幅显示。

## 其他改动

- 📘 README：用真正的 shields.io 徽章替换了手工样式的徽章（GitHub 会移除 README HTML 中的内联 `style` 属性，所以旧徽章只会显示为纯文本），更清晰的需求表格，以及明确说明 Git for Windows 是唯一需要自己安装的东西。

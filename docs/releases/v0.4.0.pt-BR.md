## Novidades

- 👥 **Aba Amigos** — o gerenciamento de amigos saiu das Configurações e ganhou uma tela própria na barra lateral. Convide pelo nome de usuário do GitHub e veja quem aceitou e quem ainda está pendente.
- 🗑️ **Excluir repositório** — uma nova ação protegida nas Configurações para excluir permanentemente o armazenamento compartilhado: diálogo de confirmação com contagem regressiva de 10 segundos antes que o botão de excluir seja liberado.

## Correções

- 🐛 Corrigido um bug de sincronização: sair de um jogo sempre enviava seu save local para o GitHub, mesmo que um amigo já tivesse enviado uma versão mais recente enquanto vocês jogavam juntos — sobrescrevendo silenciosamente o progresso dele. Agora o CoopSync verifica primeiro a versão na nuvem e pula o envio (com um aviso) se ela já for mais recente.
- 🐛 Falhas de sincronização em segundo plano antes passavam despercebidas, sem nenhum aviso — agora também aparecem em um banner.

## Outras mudanças

- 📘 README: selos estilizados manualmente substituídos por selos reais do shields.io (o GitHub remove atributos `style` inline do HTML do README, então os antigos apareciam como texto simples), uma tabela de requisitos mais clara e uma nota explícita de que o Git for Windows é a única coisa que você precisa instalar por conta própria.

A small UI polish patch — no functional changes.

## Fixes

- 🎨 **Redesigned the "last synced" / "save size" info on game cards** — it used to overflow onto a second line and break the card layout in some languages. Now it's a single compact line with icons (a clock for last sync, a disk for save size) instead of full text labels — hovering still shows the full text as a tooltip.
- 👁️ Improved the contrast of that text so it stays readable over any game cover art, no matter how bright the artwork is.
- 🐛 Fixed the "last synced" / "save size" info missing entirely for installed games listed under "All supported games" — it now shows there too, matching the "Installed games" section above.

# CoopSync v0.1.0

[English](v0.1.0.en.md) · [Français](v0.1.0.fr.md) · [Deutsch](v0.1.0.de.md) · [Polski](v0.1.0.pl.md) · **Português (Brasil)** · [Русский](v0.1.0.ru.md) · [简体中文](v0.1.0.zh-CN.md) · [Español](v0.1.0.es.md) · [Türkçe](v0.1.0.tr.md) · [Українська](v0.1.0.uk.md)

O primeiro MVP funcional do CoopSync — do zero a um aplicativo totalmente funcional em algumas sessões (29/06–01/07/2026).

## O que está incluído

- 🔑 Login com GitHub (device flow) — sem senha, sem redirecionamento OAuth
- 🔒 Repositório privado compartilhado de saves (`coopsync-saves`) — convide um amigo como colaborador, lista de membros com verificação automática do status de aceitação
- 🎮 Detecção de jogos da Steam instalados (registro + `libraryfolders.vdf`)
- 🖥️ Interface completa: integração, tela principal com cards dos jogos, configurações, barra de título e barra lateral próprias
- 🔄 Núcleo de sincronização — upload/download manual, versões dos saves, status de sincronização (synced/local-newer/remote-newer etc.)
- ⚙️ Sincronização automática — monitora a abertura/fechamento do jogo em segundo plano, pull/push automático
- 🗂️ Funcionamento em segundo plano: bandeja do sistema, inicialização automática com o Windows
- 👑 Anfitrião/entrar — escolha seu papel (sou o anfitrião / conectar a um amigo), sincronização com o repositório compartilhado
- 🎨 Sistema de design **RIFT//SYNC (Kinetic Edge)** aplicado a toda a interface

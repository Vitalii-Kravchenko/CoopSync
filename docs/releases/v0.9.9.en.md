A quick, important follow-up to yesterday's restore feature.

## Fixes

- 🛠️ **Restoring an older save version could wipe more than the save itself** — for games that keep account/platform files in the same folder as the actual save (Subnautica 2's login cache and entitlements, right next to the world save), reverting to an older version used to clear the whole folder first, taking those files with it and forcing a full game re-setup afterward. Restoring now only ever touches the actual save files, exactly like every other sync operation already did.

## Nouveautés

- 🎮 **Prise en charge de Subnautica 2** — synchronisation des sauvegardes via le dépôt GitHub privé, comme pour les autres jeux. Seuls les fichiers de sauvegarde du monde lui-même sont synchronisés (aucun fichier spécifique à la plateforme/au compte).
- ☁️ **Avertissement Steam Cloud** — l'application t'avertit si Steam Cloud risque d'entrer en conflit avec la propre synchronisation de CoopSync pour un jeu que tu synchronises.
- 🛡️ **Avertissement Windows Smart App Control** — sous Windows 11, cette fonctionnalité peut bloquer l'installation ou l'exécution d'applications non signées. Un avertissement est désormais présent dans le README, sur le premier écran de l'installateur et dans Paramètres → À propos, dans les 10 langues de l'interface.
- 🎨 Petites corrections d'interface : les icônes de notification de synchronisation sont désormais les nôtres (au lieu des flèches emoji), la version de l'application affichée dans « À propos » n'est plus codée en dur.

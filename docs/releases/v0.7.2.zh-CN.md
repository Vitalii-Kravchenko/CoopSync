一个专注于错误处理的可靠性补丁 —— 没有新功能，只是在出问题时提供更清晰的提示，并且修复了几处以前会卡住或显示误导性状态的地方。

## 修复

- 🌐 **GitHub 相关操作现在有清晰的"无网络连接"提示**（登录、创建存储、邀请好友）—— 其中一些以前会显示原始的技术错误信息。
- ⏳ 如果启动时网络检查失败，应用不再永远卡在"加载中…"—— 现在会显示清晰的错误提示，并带有"重试"按钮。
- 🔑 检查登录状态时的临时网络故障不再会把你登出 —— 现在只有真正过期的登录状态才会这样。
- 📦 设置、好友和初始设置界面不再在网络检查失败时错误地宣称存储未创建 —— 现在会显示真实的错误并提供重试选项。
- 🎮 如果加载时出了问题，游戏列表不再永远卡在"正在查找游戏…"。
- 🔄 当无法检查哪些游戏正在运行时，自动同步不再默默停止工作 —— 现在会显示提示，而不是悄悄失效。
- 📡 登录现在能挺过短暂的网络中断，而不是直接中止整个登录流程。
- ⏱️ 当 GitHub API 请求限制暂时用尽时，会出现新的提示，告诉你何时可以重试。

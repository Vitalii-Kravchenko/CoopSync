Two fixes for the update notification, so it actually reaches you and actually works when you click it later.

## Fixes

- 🔔 **The update toast reminds you again, not just once ever** — it used to share the same "once per version" rule as the notification bell, so if you missed it (or cleared everything and relaunched), you'd never see it again for that version. The bell staying a one-time record is correct; the toast now shows up again on every check — at startup, every 6 hours, or when you hit "Check for updates" — for as long as you haven't actually updated yet.
- 🔗 **Clicking the notification from the Action Center now actually opens the app** — clicking it while it was still the fresh popup already worked, but clicking that same notification later from Windows' Action Center did nothing. Windows needs an explicit app identity to route that click back to CoopSync, which is now set correctly.

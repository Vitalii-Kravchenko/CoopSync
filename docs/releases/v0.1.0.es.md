# CoopSync v0.1.0

[English](v0.1.0.en.md) · [Français](v0.1.0.fr.md) · [Deutsch](v0.1.0.de.md) · [Polski](v0.1.0.pl.md) · [Português (Brasil)](v0.1.0.pt-BR.md) · [Русский](v0.1.0.ru.md) · [简体中文](v0.1.0.zh-CN.md) · **Español** · [Türkçe](v0.1.0.tr.md) · [Українська](v0.1.0.uk.md)

El primer MVP funcional de CoopSync — de cero a una aplicación totalmente funcional en unas pocas sesiones (29/06–01/07/2026).

## Qué incluye

- 🔑 Inicio de sesión con GitHub (device flow) — sin contraseña, sin redirección OAuth
- 🔒 Repositorio privado compartido de partidas guardadas (`coopsync-saves`) — invita a un amigo como colaborador, lista de miembros con comprobación automática del estado de aceptación
- 🎮 Detección de juegos de Steam instalados (registro + `libraryfolders.vdf`)
- 🖥️ Interfaz completa: incorporación, pantalla principal con tarjetas de juegos, ajustes, barra de título y barra lateral propias
- 🔄 Núcleo de sincronización — subida/descarga manual, versiones de las partidas guardadas, estado de sincronización (synced/local-newer/remote-newer, etc.)
- ⚙️ Sincronización automática — vigila el inicio/cierre del juego en segundo plano, pull/push automático
- 🗂️ Funcionamiento en segundo plano: bandeja del sistema, inicio automático con Windows
- 👑 Anfitrión/invitado — elige tu rol (soy el anfitrión / me conecto a un amigo), sincronización con el repositorio compartido
- 🎨 Sistema de diseño **RIFT//SYNC (Kinetic Edge)** aplicado a toda la interfaz

CoopSync 的第一个可用 MVP —— 在短短几次开发中从零到一个功能完整的应用程序（2026年6月29日–7月1日）。

## 包含内容

- 🔑 GitHub 登录（设备流程）——无需密码，无需 OAuth 重定向
- 🔒 私有共享存档仓库（`coopsync-saves`）——邀请好友作为协作者，成员列表自动轮询接受状态
- 🎮 检测已安装的 Steam 游戏（注册表 + `libraryfolders.vdf`）
- 🖥️ 完整界面：引导流程、带游戏卡片的主屏幕、设置、自定义标题栏和侧边栏
- 🔄 同步核心——手动上传/下载、存档版本、同步状态（已同步/本地较新/云端较新等）
- ⚙️ 自动同步——在后台监视游戏的启动/退出，自动拉取/推送
- 🗂️ 后台运行：系统托盘、随 Windows 自动启动
- 👑 主机/加入——选择你的角色（我是主机 / 连接到朋友），与共享仓库同步
- 🎨 **RIFT//SYNC（Kinetic Edge）** 设计系统应用于整个界面

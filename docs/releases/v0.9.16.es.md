Unirse al almacenamiento de un amigo ya no requiere ir aparte a GitHub para aceptar nada.

## Correcciones

- 🤝 **Unirse ahora acepta automáticamente una invitación pendiente** — una invitación de colaborador de GitHub debe aceptarse explícitamente antes de que el acceso realmente funcione. "Conectar con un amigo" antes solo comprobaba el acceso y decía "sin acceso — pide a tu amigo que te invite", incluso justo después de que lo hubiera hecho, y reiniciar la aplicación no ayudaba porque no era un problema de caché — GitHub simplemente aún no había concedido el acceso. Ahora comprueba si hay una invitación pendiente al almacenamiento del anfitrión y la acepta automáticamente, así que unirse simplemente funciona en cuanto el anfitrión ha enviado la invitación.

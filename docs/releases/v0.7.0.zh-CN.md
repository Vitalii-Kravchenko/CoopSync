## 新功能

- 💬 **顶部栏新增"支持"按钮** — 直接在应用内发送反馈、错误报告或功能建议，无需再给我发邮件。
- 🎮 **"希望添加某个游戏"现在可以实时搜索 Steam** — 按名称（附带封面图）最多选择 3 个游戏，而不用输入自由文本，让请求更清晰、更容易处理。
- 💡 **新增"有个想法"分类** — 用于错误报告和游戏请求之外的任何内容。
- ✉️ 反馈邮件现在会显示你的 GitHub 名称、可点击的所请求游戏的 Steam 链接，整体也更美观了。

## 修复

- ⌨️ **无障碍性修复** — 对话框（支持、删除确认）现在能正确地将键盘焦点保持在自身内部，而不是"泄漏"到后面的应用界面，焦点指示器现在在各处都能正常工作，包括应用标志性的斜切角按钮。

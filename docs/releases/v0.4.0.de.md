## Neu

- 👥 **Freunde-Tab** — die Freundeverwaltung ist aus den Einstellungen in einen eigenen Bildschirm in der Seitenleiste umgezogen. Lade per GitHub-Benutzername ein und sieh, wer angenommen hat und wer noch aussteht.
- 🗑️ **Repository löschen** — eine neu abgesicherte Aktion in den Einstellungen, um den gemeinsamen Speicher endgültig zu löschen: Bestätigungsdialog mit 10-Sekunden-Countdown, bevor die Löschen-Schaltfläche freigeschaltet wird.

## Behoben

- 🐛 Sync-Fehler behoben: Beim Beenden eines Spiels wurde der lokale Spielstand immer auf GitHub gepusht, selbst wenn ein Freund bereits eine neuere Version gepusht hatte, während ihr zusammen gespielt habt — das hat seinen Fortschritt stillschweigend überschrieben. CoopSync prüft jetzt zuerst die Cloud-Version und überspringt den Push (mit Warnhinweis), falls sie bereits neuer ist.
- 🐛 Fehlgeschlagene Hintergrund-Synchronisierungen scheiterten bisher stillschweigend ohne Rückmeldung — werden jetzt ebenfalls als Banner angezeigt.

## Weitere Änderungen

- 📘 README: handgestaltete Badges durch echte shields.io-Badges ersetzt (GitHub entfernt Inline-`style`-Attribute aus README-HTML, daher wurden die alten als reiner Text dargestellt), eine klarere Anforderungstabelle und ein expliziter Hinweis, dass Git for Windows das Einzige ist, was man selbst installieren muss.

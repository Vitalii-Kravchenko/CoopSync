# CoopSync v0.6.0

**English** · [Français](v0.6.0.fr.md) · [Deutsch](v0.6.0.de.md) · [Polski](v0.6.0.pl.md) · [Português (Brasil)](v0.6.0.pt-BR.md) · [Русский](v0.6.0.ru.md) · [简体中文](v0.6.0.zh-CN.md) · [Español](v0.6.0.es.md) · [Türkçe](v0.6.0.tr.md) · [Українська](v0.6.0.uk.md)

## New

- 🎮 **Redesigned game card hover** — hovering an installed game now shows full-size Upload/Download buttons with labels, plus the last sync time and save size, instead of two small icon-only buttons.
- ✨ Unified hover and transition timings across the whole app — cards, buttons, toggles, and the sidebar now move at consistent speeds instead of slightly different ones.

## Fixes

- 🌍 **All app error messages are now fully localized** — errors from GitHub, sync, and storage used to always show in Ukrainian no matter which language you'd picked. Every message is now translated into all 10 supported languages.
- 🔇 Fixed several places where a failed action showed nothing at all — inviting a friend, loading sync history, and checking sync status could fail with zero feedback. They now show a clear error message (and, for sync status, a "Try again" button).
- 🔁 The app now automatically recovers if its local sync folder goes stale (e.g. after the shared repository was deleted and re-created), instead of failing forever with a technical git error.
- 🐛 Fixed a mix-up where inviting a nonexistent GitHub username showed "storage not found" instead of "user not found."
- ➕ Settings now shows a "Create storage" button when no shared storage is set up (e.g. right after deleting it) — it used to just say "storage not set up" with no way to fix it from there.

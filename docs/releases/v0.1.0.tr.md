İlk çalışan CoopSync MVP'si — birkaç oturumda sıfırdan tamamen işlevsel bir uygulamaya (29.06–01.07.2026).

## Neler var

- 🔑 GitHub girişi (device flow) — şifre yok, OAuth yönlendirmesi yok
- 🔒 Özel paylaşılan kayıt deposu (`coopsync-saves`) — bir arkadaşı katkıda bulunan olarak davet et, kabul durumunu otomatik kontrol eden üye listesi
- 🎮 Kurulu Steam oyunlarının tespiti (kayıt defteri + `libraryfolders.vdf`)
- 🖥️ Tam arayüz: onboarding, oyun kartlarıyla ana ekran, ayarlar, özel başlık çubuğu ve kenar çubuğu
- 🔄 Senkronizasyon çekirdeği — manuel yükleme/indirme, kayıt sürümleri, senkronizasyon durumu (synced/local-newer/remote-newer vb.)
- ⚙️ Otomatik senkronizasyon — arka planda oyunun açılışını/kapanışını izler, otomatik pull/push
- 🗂️ Arka planda çalışma: sistem tepsisi, Windows ile otomatik başlatma
- 👑 Host/join — rolünü seç (ben ana bilgisayarım / bir arkadaşa bağlanıyorum), paylaşılan depoyla senkronizasyon
- 🎨 **RIFT//SYNC (Kinetic Edge)** tasarım sistemi tüm arayüze uygulandı

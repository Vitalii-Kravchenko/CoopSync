A data-loss fix — restoring while the game hadn't fully exited yet could quietly wipe out the session you just played.

## Fixes

- 🛡️ **Restore now checks the game is actually closed** — restoring an older save version while the game process was still running (even just lingering right after you quit) overwrote your just-played local saves with the old version. Since Restore also pushes right away, the background autopush that ran moments later found nothing left to upload — silently losing that session for good. Restore now refuses with a clear message if the game hasn't fully exited yet.

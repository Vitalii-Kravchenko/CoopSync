Sieben Korrekturen, die sich aus dem täglichen Gebrauch benutzerdefinierter Spiele ergeben haben.

## Korrekturen

- 🌐 **Der Installer spricht jetzt deine Windows-Sprache** — der Smart-App-Control-Hinweis vor der Installation wurde bisher immer auf Englisch angezeigt, obwohl er in alle 10 Sprachen übersetzt ist. Er passt sich jetzt automatisch an deine Windows-Anzeigesprache an und fällt auf Englisch zurück, falls deine nicht dabei ist.
- 🎯 **Die gewählte .exe eines benutzerdefinierten Spiels bleibt jetzt tatsächlich ausgewählt** — beim erneuten Öffnen der Detailansicht wurde gar keine ausgewählte Datei angezeigt, obwohl sie korrekt gespeichert war. Nur ein Anzeigefehler, der den Eindruck erweckte, die Auswahl sei verloren.
- 🏷️ **"Manuell hinzugefügt" steht jetzt neben dem Spielnamen** statt in einer eigenen Zeile darunter.
- 🖼️ **Dateiauswahl-Dialoge (Cover, Profilbild) folgen jetzt deiner gewählten Sprache** — bisher wurden sie immer auf Ukrainisch angezeigt, unabhängig von der Einstellung.
- 🙂 **Dein eigenes Profilbild übersteht jetzt eine komplette Neuinstallation** — es verschwand bisher überall (bei dir und deinem Freund), sobald du deinstalliert und neu installiert hast, obwohl es sicher auf GitHub lag. Wird jetzt automatisch wiederhergestellt.
- 🗂️ **Dateien direkt beim Hinzufügen eines benutzerdefinierten Spiels ausschließen** — nicht mehr erst hinzufügen und die Option danach in den Details suchen.
- ☁️ **Wenn ein Cover nicht bei GitHub ankommt, wird es jetzt gemeldet** — das Hinzufügen eines benutzerdefinierten Spiels oder das Ändern seines Covers konnte bisher lautlos fehlschlagen, ohne dass dein Mitspieler es je sah. Jetzt gibt es eine Warnung mit einer "Erneut versuchen"-Schaltfläche, statt dass es unbemerkt verschwindet.

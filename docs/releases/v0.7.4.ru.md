Патч согласованности дизайна и клавиатурной доступности — интерфейс теперь единообразен на всех вкладках, а управление с клавиатуры работает одинаково везде.

## Исправления

- 🎯 **Фокус клавиатуры (Tab) теперь виден везде.** На нескольких элементах (кнопки окна, пункты бокового меню, некоторые ссылки) фокус раньше не отображался вовсе, либо показывалась обычная жёлтая рамка Windows вместо фирменного циан-кольца приложения.
- ⌨️ Переключатели в Настройках (автозапуск, сворачивание в трей, предупреждение Steam Cloud) теперь полностью управляются с клавиатуры — раньше они реагировали только на мышь.
- 🌐 Выбор языка в Настройках теперь собственный выпадающий список в стиле приложения, а не обычный системный список Windows.
- 🗂️ Порядок перехода по Tab теперь логичен: сначала контент текущей вкладки и боковая навигация, и только потом кнопка "Поддержка" и кнопки окна (свернуть/восстановить/закрыть).
- 🎮 На карточках игр кнопки "Загрузить"/"Скачать" (видны только при наведении) больше не "съедают" Tab в большой библиотеке — Tab теперь просто проходит карточку за карточкой.
- 🖼️ Заглушка аватарки (пока фото ещё не задано) теперь одинакова везде — в верхней панели, Настройках, онбординге и списке друзей.
- 🎨 Небольшие визуальные правки по всему приложению: бейджи статуса синхронизации, предупреждения, карточки недоступных игр и ещё несколько элементов теперь выглядят одинаково.

## Fixes

- 🐛 Fixed: a game that's both supported and already installed used to disappear entirely from "All supported games" (that section only showed games you hadn't installed yet) — confusing, since it looked unsupported. Now it shows there too, with full sync controls.
- ⚠️ The Steam Cloud warning banner now mentions that you can turn it off in Settings → General.

## Other changes

- 📄 License changed from MIT to **PolyForm Noncommercial 1.0.0** — CoopSync stays free to use, copy and modify, but can no longer be resold or used commercially.
- 📘 README: added badges, a Requirements section (Windows, Steam, system Git in PATH, a GitHub account), and an IMPORTANT note about the Steam Cloud conflict.

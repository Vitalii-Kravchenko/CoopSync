# CoopSync v0.6.0

## 新功能

- 🎮 **重新设计的游戏卡片悬停效果** — 将鼠标悬停在已安装的游戏上时，现在会显示带文字标签的完整大小的上传/下载按钮，以及上次同步时间和存档大小，而不再是两个只有图标的小按钮。
- ✨ 统一了整个应用的悬停和过渡动画时长 — 卡片、按钮、开关和侧边栏现在以一致的速度移动，而不是各自略有差异。

## 修复

- 🌍 **所有应用错误消息现在都已完全本地化** — 以前无论选择哪种语言，来自 GitHub、同步和存储的错误始终以乌克兰语显示。现在每条消息都已翻译成全部 10 种支持的语言。
- 🔇 修复了几处操作失败时完全没有任何提示的问题 — 邀请好友、加载同步历史和检查同步状态都可能在毫无反馈的情况下失败。现在它们会显示清晰的错误信息（同步状态还会额外显示一个"重试"按钮）。
- 🔁 当本地同步文件夹过期时（例如共享仓库被删除并重新创建后），应用现在会自动恢复，而不是一直卡在一个技术性的 git 错误上。
- 🐛 修复了一处混淆：邀请一个不存在的 GitHub 用户名时会显示"未找到存储"，而不是"未找到用户"。
- ➕ 当没有配置共享存储时（例如刚删除之后），设置界面现在会显示一个"创建存储"按钮 — 以前那里只写着"存储未设置"，却没有任何办法在那里直接解决。

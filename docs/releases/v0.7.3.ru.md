Патч на надёжность — самая важная часть исправляет два бага, которые могли навсегда сломать синхронизацию у друга. Также добавлено: общие аватарки.

## Новое

- 🖼️ **Аватарки теперь синхронизируются между тобой и другом.** Поставь картинку в Настройках — она загрузится в общее хранилище, и твой напарник тоже её увидит (а ты — его), вместо обычной заглушки.

## Исправления

- 🔧 **Критично: синхронизация могла сломаться навсегда.** Если приложение закрывалось или падало посреди выгрузки сейва, внутренняя рабочая копия могла остаться в "грязном" состоянии — любая синхронизация после этого (история, выгрузка, загрузка) падала с ошибкой "local changes would be overwritten by merge" без возможности восстановиться без ручного вмешательства. Теперь приложение само сбрасывает это внутреннее состояние перед каждой синхронизацией.
- 🔑 **Критично: выгрузка могла не сработать вообще** с ошибкой "unable to auto-detect email address" на машине, где git никогда не настраивали именем/почтой. Теперь приложение само задаёт идентичность коммита, не полагаясь на глобальный git-конфиг машины.
- 👤 История синхронизации и "последний раз обновил" теперь правильно показывают, кто реально выполнил действие — раньше всегда показывало владельца хранилища, даже когда синхронизировал именно твой напарник.
- 👤 В списке участников (когда ты присоединился к хранилищу друга) строка "владелец" показывала твой собственный ник вместо настоящего владельца — теперь всегда показывает реального. Заглушка аватарки (когда картинки ещё нет) теперь тоже одинаковая везде.
- 🔄 Вкладка "Друзья" больше не показывает устаревшие данные — она перечитывается автоматически каждый раз, когда ты на неё заходишь, а не только один раз при старте приложения.
- 🪟 Повторный запуск приложения (напр. двойной клик по ярлыку, пока оно уже сидит в трее) больше не открывает второй процесс — просто поднимает уже открытое окно.
- 🧭 Исправлено смещение предупреждения "нет доступа к хранилищу" на экране подключения — оно было сдвинуто влево вместо того, чтобы выравниваться с остальными шагами настройки.

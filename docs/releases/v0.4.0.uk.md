## Нове

- 👥 **Вкладка «Друзі»** — керування друзями переїхало з Налаштувань в окремий екран у сайдбарі. Запрошуй за ніком GitHub, дивись хто прийняв запрошення, а хто ще очікує.
- 🗑️ **Видалення репозиторію** — нова небезпечна дія в Налаштуваннях: назавжди видалити спільне сховище сейвів, з підтвердженням і 10-секундним відліком, перш ніж кнопка розблокується.

## Виправлення

- 🐛 Виправлено баг синку: вихід з гри завжди пушив локальний сейв на GitHub, навіть якщо друг уже запушив новішу версію, поки ви грали разом — це мовчки затирало його прогрес. Тепер CoopSync спершу перевіряє версію в хмарі і пропускає push (з попередженням), якщо вона вже новіша.
- 🐛 Невдалі фонові синхронізації раніше мовчки губились без жодного повідомлення — тепер теж показуються банером.

## Інші зміни

- 📘 README: замінено самописні бейджі на справжні shields.io (GitHub вирізає інлайн `style` з HTML у README, тож старі рендерились голим текстом), чіткіша таблиця вимог і явна примітка, що Git для Windows — єдине, що треба ставити самому.

一次设计一致性与键盘可访问性补丁——界面在所有标签页中现在保持一致，键盘导航在各处的表现也完全相同。

## 修复

- 🎯 **键盘焦点（Tab）现在处处可见。** 此前有几个元素（窗口按钮、侧边栏导航项、部分链接）完全不显示焦点指示，或者显示的是 Windows 通用的黄色轮廓，而不是应用自己的青色焦点环。
- ⌨️ 设置中的开关（开机自启、以最小化方式启动、Steam Cloud 提醒）现在完全可以用键盘操作——此前它们只能响应鼠标点击。
- 🌐 设置中的语言选择现在是应用风格的自定义下拉菜单，而不是普通的 Windows 系统列表。
- 🗂️ Tab 键的切换顺序现在更符合逻辑：先是当前标签页的内容和侧边栏导航，之后才是"支持"按钮和窗口按钮（最小化/还原/关闭）。
- 🎮 在游戏卡片上，"上传"/"下载"按钮（仅在鼠标悬停时可见）不再在大型游戏库中"吞掉"Tab 键——现在 Tab 会直接从一张卡片跳到下一张。
- 🖼️ 默认头像（在设置图片之前）现在在各处保持一致——标题栏、设置、初始引导流程和好友列表。
- 🎨 应用中多处细小的视觉一致性修复：同步状态徽章、警告横幅、不可用游戏卡片以及其他一些元素现在在各处外观统一。

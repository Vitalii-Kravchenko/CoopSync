Primeiro passo rumo a um histórico de restauração próprio para cada jogo.

## Novidades

- 🕓 Cada card de jogo agora tem um botão **Detalhes** que abre uma tela dedicada com o histórico de sincronização daquele jogo — separado do histórico compartilhado na aba "Histórico".
- 🧭 Navegação em breadcrumb (`Jogos / Nome do jogo → Histórico de sincronização`) para voltar à biblioteca.

Restaurar uma versão anterior do save a partir dessa tela chega em uma atualização futura — por enquanto é só uma visualização do histórico, sem edição.

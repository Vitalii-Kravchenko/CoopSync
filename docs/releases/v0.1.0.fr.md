Le premier MVP fonctionnel de CoopSync — de zéro à une application pleinement fonctionnelle en quelques sessions (29/06–01/07/2026).

## Ce qui est inclus

- 🔑 Connexion GitHub (device flow) — sans mot de passe, sans redirection OAuth
- 🔒 Dépôt privé partagé pour les sauvegardes (`coopsync-saves`) — invite un ami en tant que collaborateur, liste des membres avec vérification automatique du statut d'acceptation
- 🎮 Détection des jeux Steam installés (registre + `libraryfolders.vdf`)
- 🖥️ Interface complète : intégration, écran principal avec les cartes des jeux, paramètres, barre de titre et barre latérale personnalisées
- 🔄 Cœur de synchronisation — envoi/téléchargement manuel, versions des sauvegardes, statut de synchronisation (synced/local-newer/remote-newer, etc.)
- ⚙️ Synchronisation automatique — surveille le lancement/la fermeture du jeu en arrière-plan, pull/push automatique
- 🗂️ Fonctionnement en arrière-plan : icône dans la barre d'état, démarrage automatique avec Windows
- 👑 Hôte/invité — choisis ton rôle (je suis l'hôte / je me connecte à un ami), synchronisation avec le dépôt partagé
- 🎨 Système de design **RIFT//SYNC (Kinetic Edge)** appliqué à toute l'interface

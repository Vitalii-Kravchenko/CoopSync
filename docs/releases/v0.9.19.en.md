A small move for a clearer app: connecting to a friend now lives where you'd actually look for it.

## Changed

- 🤝 **"Connect to a friend" moved from Settings to the Friends tab** — last release added a way to accept an invite or type a friend's username without leaving the app, but it lived in Settings' Storage card. Friends is where you'd actually think to look for it, and it already doubles as "your co-op group" screen regardless of role. Its old placeholder message ("create storage in Settings first") is now the real thing: the same one-click invite banner and username field, right there. Creating your own storage stays in Settings, since that's a repo-ownership decision.

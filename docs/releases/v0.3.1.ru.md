## Исправления

- 🐛 Исправлено: игра, которая одновременно поддерживается и уже установлена, полностью пропадала из списка "Все поддерживаемые игры" (там показывались только неустановленные) — это сбивало с толку, будто игра не поддерживается. Теперь она показывается и там тоже, с полноценными кнопками синка.
- ⚠️ Предупреждение о Steam Cloud теперь упоминает, что его можно отключить в Настройках → Общее.

## Другие изменения

- 📄 Лицензия изменена с MIT на **PolyForm Noncommercial 1.0.0** — CoopSync остаётся бесплатным для использования, копирования и модификации, но больше нельзя перепродавать или использовать коммерчески.
- 📘 README: добавлены бейджи, раздел требований (Windows, Steam, системный Git в PATH, аккаунт GitHub) и примечание IMPORTANT о конфликте со Steam Cloud.

## Fehlerbehebungen

- 🎨 **Die Anzeige "Zuletzt synchronisiert" / "Speichergröße" auf den Spielkarten überarbeitet** — sie ist früher in manchen Sprachen in eine zweite Zeile umgebrochen und hat das Kartenlayout zerstört. Jetzt ist es eine kompakte Zeile mit Symbolen (eine Uhr für die letzte Synchronisierung, eine Diskette für die Speichergröße) statt vollem Text — beim Hovern wird der vollständige Text weiterhin als Tooltip angezeigt.
- 👁️ Kontrast dieses Textes verbessert, damit er über jedem Spiel-Cover lesbar bleibt, egal wie hell das Artwork ist.
- 🐛 Behoben, dass "Zuletzt synchronisiert" / "Speichergröße" bei installierten Spielen unter "Alle unterstützten Spiele" komplett fehlte — wird jetzt dort genauso angezeigt wie im Abschnitt "Installierte Spiele" darüber.

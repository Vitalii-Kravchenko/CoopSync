## 更新内容

- 🎮 **支持 Subnautica 2** —— 通过私有 GitHub 仓库同步存档，与其他游戏相同。仅同步游戏世界本身的存档文件（不包含平台/账户特定文件）。
- ☁️ **Steam Cloud 提示** —— 如果 Steam Cloud 可能与 CoopSync 对某个游戏的同步产生冲突，应用会提示你。
- 🛡️ **Windows Smart App Control 提示** —— 在 Windows 11 上，此功能可能会阻止安装或运行未签名的应用程序。现在 README、安装程序的第一个界面以及"设置 → 关于"中都有此提示，支持全部 10 种界面语言。
- 🎨 小的界面修复：同步通知图标现在是我们自己的（而非表情符号箭头），"关于"中显示的应用版本不再是硬编码的。

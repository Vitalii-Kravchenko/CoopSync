Entrar no armazenamento de um amigo não exige mais uma ida separada ao GitHub para aceitar alguma coisa.

## Correções

- 🤝 **Entrar agora aceita automaticamente um convite pendente** — um convite de colaborador do GitHub precisa ser explicitamente aceito antes que o acesso realmente funcione. "Conectar a um amigo" antes só verificava o acesso e dizia "sem acesso — peça para seu amigo te convidar", mesmo logo depois de ele ter feito isso, e reiniciar o app não ajudava, já que não era um problema de cache — o próprio GitHub ainda não tinha concedido o acesso. Agora ele verifica se há um convite pendente para o armazenamento do host e o aceita automaticamente, então entrar simplesmente funciona assim que o host enviar o convite.

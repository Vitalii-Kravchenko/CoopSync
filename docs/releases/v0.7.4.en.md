A design-consistency and keyboard-accessibility patch — the interface is now consistent across every tab, and keyboard navigation works the same way everywhere.

## Fixes

- 🎯 **Keyboard focus (Tab) is now visible everywhere.** Several elements (window buttons, sidebar nav items, some links) used to show no focus indicator at all, or showed Windows' generic yellow outline instead of the app's own cyan focus ring.
- ⌨️ Toggles in Settings (autostart, start minimized, Steam Cloud warning) are now fully keyboard-operable — they used to be visual switches that only responded to mouse clicks.
- 🌐 The language picker in Settings is now a custom dropdown matching the app's style, instead of a plain Windows system list.
- 🗂️ Tab order is now logical: the current tab's content and the sidebar come first, and only after that the Support button and window buttons (minimize/restore/close).
- 🎮 On game cards, the Upload/Download buttons (only visible on hover) no longer "eat" Tab presses in a large library — Tab now moves straight from card to card.
- 🖼️ The placeholder avatar (before a picture is set) is now the same everywhere — title bar, Settings, onboarding, and the friends list.
- 🎨 Small visual consistency fixes across the app: sync-status badges, warning banners, disabled game cards, and a few other elements now look the same everywhere.

Erster Schritt zu einer Rollback-Historie für jedes Spiel einzeln.

## Neu

- 🕓 Jede Spielkarte hat jetzt eine Schaltfläche **Details**, die einen eigenen Bildschirm mit der Sync-Historie genau dieses Spiels öffnet — getrennt von der gemeinsamen Historie im Tab "Verlauf".
- 🧭 Breadcrumb-Navigation (`Spiele / Spielname → Sync-Historie`), um zurück zur Bibliothek zu gelangen.

Das Zurücksetzen auf eine ältere Speicherstand-Version von diesem Bildschirm aus kommt in einem zukünftigen Update — vorerst ist es nur eine reine Verlaufsansicht.

一个修复：一些安装因为时间推移积累下来的重复开机启动项。

## 修复

- 🚀 **CoopSync 不会再随 Windows 启动两次了。** 较早的版本（在应用还没有为 Windows 设置明确身份标识之前）用一个默认的、自动生成的名称注册了开机启动项。后来的版本改为用应用真正的身份标识来管理开机启动，却从未清理过那个旧的启动项——于是每次登录时两个都会触发，多启动出一个多余的进程（虽然会被立即关闭，但仍然会在 Windows 的启动应用列表和任务管理器里显示为多余的一项）。这次更新会在首次启动时自动移除这个过时的启动项。

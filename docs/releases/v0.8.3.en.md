A couple of small follow-up fixes.

## Fixes

- 🕓 "Check for updates" in Settings could get stuck on "Checking for updates..." forever if the check never came back — it now times out after 20 seconds and shows an error instead, so the button always recovers.
- 🎯 The sync-status badge's dot and text (touched up in 0.8.2) still sat a hair off each other in some cases — nudged the text down slightly so it's properly centered now.

## 修复

- 🐛 修复：一个既受支持又已安装的游戏会从"所有支持的游戏"列表中完全消失（那里之前只显示未安装的游戏）——这很令人困惑，看起来像是不受支持。现在它也会显示在那里，并带有完整的同步控件。
- ⚠️ Steam Cloud 提示现在会说明可以在 设置 → 通用 中将其关闭。

## 其他更改

- 📄 许可证从 MIT 更改为 **PolyForm Noncommercial 1.0.0** —— CoopSync 仍然可以免费使用、复制和修改，但不能再转售或用于商业用途。
- 📘 README：添加了徽章、一个"要求"部分（Windows、Steam、PATH 中的系统 Git、一个 GitHub 账户），以及关于 Steam Cloud 冲突的重要提示。

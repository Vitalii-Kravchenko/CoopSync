## Fixes

- 🤝 **A custom game's cover now really reaches your co-op partner** — it was landing correctly on GitHub the whole time, but their app wasn't refreshing its games list to pick it up unless a brand-new game happened to show up in the same check. Now it always refreshes.
- 🖼️ **A custom game's cover now shows up in the History tab too** — it used to always stay blank there, even when the game's own detail screen displayed it just fine.
- 📝 **A note on excluding files** — added a line explaining that file exclusions are per-PC only, not shared with your co-op partner (the same "junk" file can have a different name on each install, so a copied list wouldn't reliably match anyway). Each person sets this up separately, based on what they see in their own save folder.

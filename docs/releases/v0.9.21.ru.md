Теперь можно синхронизировать любую свою игру — не только те, что уже есть в каталоге, и не только по угаданному пути по умолчанию.

## Новое

- 🗂️ **Редактирование папки сохранений любой игры прямо с экрана деталей** — если CoopSync угадал путь неправильно (установка не через Steam, перенесённая папка пользователя и т.д.), теперь это можно исправить самому. Открой экран деталей игры → «Расположение сохранений» → Редактировать, укажи реальную папку, сохрани. «Сбросить по умолчанию» вернёт угаданный каталогом путь.
- ➕ **Добавление игры вне встроенного каталога** — карточка «Добавить игру» теперь сверху списка установленных игр. Укажи папку сохранений и, по желанию, папку установки игры — CoopSync сам просканирует её и найдёт .exe, включив ту же автосинхронизацию при запуске/выходе, что и у игр из каталога (или укажи .exe вручную, если сканирование что-то пропустило). Без папки установки тоже работает — просто Upload/Download вручную. Поскольку структура сохранений своей игры неизвестна, синхронизируется вся папка как есть, без фильтрации, в отличие от игр из каталога.
- 🤝 **Добавленная вручную игра появляется и у напарника** — она попадает в небольшой общий реестр в репозитории, поэтому приложение напарника само её подхватывает и предлагает указать свою папку сохранений, прежде чем синхронизация заработает и у него.

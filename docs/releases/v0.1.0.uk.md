# CoopSync v0.1.0

[English](v0.1.0.en.md) · [Français](v0.1.0.fr.md) · [Deutsch](v0.1.0.de.md) · [Polski](v0.1.0.pl.md) · [Português (Brasil)](v0.1.0.pt-BR.md) · [Русский](v0.1.0.ru.md) · [简体中文](v0.1.0.zh-CN.md) · [Español](v0.1.0.es.md) · [Türkçe](v0.1.0.tr.md) · **Українська**

Перший робочий MVP CoopSync — з нуля до повністю функціонального застосунку за кілька занять (29.06–01.07.2026).

## Що є

- 🔑 Логін через GitHub (device flow) — без пароля, без OAuth-редіректу
- 🔒 Приватне спільне сховище сейвів (`coopsync-saves`) — запрошення друга у співавтори, список учасників з автополінгом статусу прийняття
- 🎮 Детект встановлених Steam-ігор (реєстр + `libraryfolders.vdf`)
- 🖥️ Повний UI: онбординг, головний екран з картками ігор, налаштування, власний titlebar і бічна панель
- 🔄 Ядро синхронізації — ручний upload/download, версії сейвів, статус синку (synced/local-newer/remote-newer тощо)
- ⚙️ Автосинхронізація — стеження за запуском/виходом гри у фоні, авто-pull/push
- 🗂️ Робота у фоні: трей, автозапуск разом із Windows
- 👑 Host/join — вибір ролі (я головний / підключаюсь до друга), синхронізація зі спільним сховищем
- 🎨 Дизайн-система **RIFT//SYNC (Kinetic Edge)** застосована на весь інтерфейс

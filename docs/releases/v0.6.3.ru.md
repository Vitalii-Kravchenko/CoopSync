Небольшой патч с полировкой UI — без функциональных изменений.

## Исправления

- 🎨 **Переработана надпись "последняя синхронизация" / "размер сейвов" на карточках игр** — раньше она переносилась на вторую строку и ломала верстку карточки на некоторых языках. Теперь это одна компактная строка с иконками (часы для времени синка, дискета для размера) вместо полного текста — при наведении курсора полный текст по-прежнему показывается подсказкой.
- 👁️ Улучшен контраст этого текста, чтобы он читался на любой обложке игры, независимо от того, насколько яркая там картинка.
- 🐛 Исправлено полное исчезновение надписи "последняя синхронизация" / "размер сейвов" для установленных игр в списке "Все поддерживаемые игры" — теперь она показывается и там, так же как в разделе "Установленные игры" выше.

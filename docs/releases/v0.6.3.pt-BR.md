Um pequeno patch de polimento de UI — sem mudanças funcionais.

## Correções

- 🎨 **Redesenhada a informação "última sincronização" / "tamanho dos saves" nos cards de jogo** — antes ela quebrava para uma segunda linha e bagunçava o layout do card em alguns idiomas. Agora é uma única linha compacta com ícones (um relógio para a última sincronização, um disquete para o tamanho) em vez de texto completo — passar o mouse ainda mostra o texto completo como dica.
- 👁️ Melhorado o contraste desse texto para continuar legível sobre qualquer capa de jogo, não importa o quão clara seja a arte.
- 🐛 Corrigida a ausência total da informação "última sincronização" / "tamanho dos saves" para jogos instalados listados em "Todos os jogos suportados" — agora aparece lá também, igual à seção "Jogos instalados" acima.

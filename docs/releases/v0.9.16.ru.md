Подключение к хранилищу друга больше не требует отдельно заходить на GitHub, чтобы что-то там принять.

## Исправления

- 🤝 **Подключение теперь само принимает приглашение, ожидающее подтверждения** — приглашение в соавторы на GitHub нужно явно принять, прежде чем доступ реально заработает. «Подключиться к другу» раньше просто проверяло доступ и писало «нет доступа — пусть друг тебя пригласит», даже если друг только что это сделал, и перезапуск программы не помогал, потому что это не проблема кэша — сам GitHub ещё не выдал доступ. Теперь программа проверяет, есть ли приглашение в хранилище хоста, ожидающее подтверждения, и принимает его автоматически — так что подключение просто работает, как только хост отправил приглашение.

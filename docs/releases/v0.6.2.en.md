Another reliability patch — no new features, just fixes around deleting/recreating the shared storage, keeping sync status and history fresh, and a bit of UI polish.

## Fixes

- 🗑️ **Deleting and recreating the shared saves repository no longer breaks the app** — game cards, sync status, and history now correctly reset and catch up to the current state instead of getting stuck on stale data.
- 🔌 A deleted repository now shows a clear "Repository not connected" status on each game card, instead of hanging forever on "Checking…".
- ✅ **Auto-sync no longer silently dies** after you delete and recreate the storage — previously this required restarting the app to work again.
- 💬 Fixed a misleading "Uploaded to GitHub" message that appeared even when nothing had actually changed — it now honestly says "No changes — nothing to upload".
- 🔔 The sync notification banner is now visible from any tab, not just the Games screen.
- 🔄 Sync status and history now refresh automatically when you switch to the Games or History tab, and right after any push or pull — not only at app launch.
- ✨ New entries in the sync history animate in exactly once, right when they actually become visible — no more duplicated or delayed "just arrived" effects when switching tabs.
- 📏 Widened the delete-repository confirmation dialog so the button label and countdown never wrap onto two lines, in any language.
